# Clickable Transparent Overlay
A library for creating transparent overlay using Win32 API, Threads/Tasks and ImGui.NET.

# Nuget
https://www.nuget.org/packages/ClickableTransparentOverlay

# Dependencies
* .NET Core 2.0
